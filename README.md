# Web Scraping Tools

A collection of web scrapers for Amazon, eBay, and general-purpose scraping.

## Included Scrapers

- **AmazonSearch** — Scrape Amazon product search results
- **eBay** — Scrape eBay product listings and details  
- **WebScrapProject** — General-purpose web scraping utility

## Tech Stack

- **Language**: Python
- **Libraries**: requests, BeautifulSoup, pandas

## Getting Started

```bash
pip install requests beautifulsoup4 pandas
python AmazonSearch/main.py
python ebay/main.py
```
